# FriendsOfGlobalAuth Brand Style

Brand colors are derived from a corporate palette.

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Dark background | `#0e131b` | Shield fill, dark surfaces |
| Dark secondary | `#1a212e` | Shield gradient top, cards |
| Green primary | `#3f9126` | Borders, primary people, action elements |
| Green light | `#69b449` | Globe lines, secondary people, lighter accents |
| Gold primary | `#ffc759` | Lock, highlights |
| Gold secondary | `#f9af1e` | Lock gradient end, warm accents |
| White | `#ffffff` | Text on dark backgrounds |

## Logo

Source: `assets/logo.svg`

Elements: dark shield on light circle, "GA" text, globe with meridians, gold lock, three green people silhouettes (friends/community).

## Usage

When generating UI, documentation graphics, or presentation materials for FriendsOfGlobalAuth — use these colors to maintain visual consistency.
